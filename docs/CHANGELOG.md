# Changelog

All notable changes to DeepReport will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Multi-agent collaboration framework with Planning Agent
- DeepResearcherAgent for high-quality source filtering
- BrowserAgent with browser-use library integration
- PDF analysis capabilities with PyMuPDF
- Multi-engine search integration (Serper, Metaso, Sogou)
- MCP protocol support using FastMCP
- HTML report generation with interactive charts
- Professional citation management (APA, MLA, Chicago, Harvard)
- Gradio web interface
- Comprehensive documentation and contribution guidelines

### Changed
- Improved agent architecture with better task decomposition
- Enhanced browser interaction capabilities
- Better error handling and logging
- Optimized report generation performance

### Fixed
- Initial release bug fixes
- Improved dependency management
- Better configuration handling

## [1.0.0] - 2024-01-XX

### Added
- Initial release of DeepReport

- HTML report generation
- Gradio interface
- MCP protocol support
- Documentation and setup guides

## [0.1.0] - 2024-01-XX

### Added
